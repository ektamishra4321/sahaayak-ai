## What SAFE-T Covers
SAFE-T (Seller Assurance for e-Commerce Transactions) lets a seller request
reimbursement from Amazon when a refund or replacement was granted to a buyer
and the seller believes the loss was not their fault - for example the buyer
returned a different, used, or damaged item, the item was damaged in transit
under Easy Ship, or a refund was issued without the item coming back. It
applies to seller-fulfilled programs (Easy Ship, Self-Ship, Seller Flex).
FBA inventory issues follow a separate reimbursement process, not SAFE-T.

## Deadlines (program-specific - always check eligibility in Seller Central)
Deadlines vary by scenario and have been revised recently, so treat these as
indicative [VERIFY each in the Seller Central eligibility checker]:
- Damage-related claims for Easy Ship shipments: file within about 15
  calendar days of the relevant shipment scan/event.
- Return received in bad condition: file promptly after the return arrives -
  do not wait; late claims are auto-rejected.
- Return order not received: filed in a later window (only after enough days
  have passed for the return to arrive, but before the outer deadline shown
  in the checker).
- Appeals: allowed once per claim, within roughly 5-7 days of the decision
  [VERIFY - sources differ]; the exact "Appeal by" date is shown on the
  Manage SAFE-T Claims page against each claim.

## How to File
1. Seller Central > Orders > Manage SAFE-T Claims > File a new SAFE-T claim.
2. Enter the Order ID and click Check Eligibility (the Seller App can also
   scan the RMA to check instantly). If ineligible, the tool says so and no
   claim can be filed for that order.
3. Select the reason code that best matches, attach evidence, describe the
   issue specifically, and submit. Amazon confirms receipt by email and
   usually responds within about 7 business days.
4. If Amazon's SAFE-T team asks follow-up questions, reply within the stated
   response window shown on the claim [VERIFY - roughly 3 working days per
   India seller guides] or the claim closes automatically.

## Evidence That Wins Claims
Photograph every return on opening (all angles, packaging, label), keep the
invoice and dispatch-weight records, and cite specific discrepancies - serial
number mismatch, missing accessories, different brand/model - rather than
general statements like "item is bad". Reimbursement is based on fair market
value as determined by Amazon, with applicable fees deducted, so accurate
cost/invoice proof matters.
